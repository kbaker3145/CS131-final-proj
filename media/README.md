## TEST: threedirtydishes
This test is a very simple test of our system. We have two videos in this folder: one of the face camera setup and one of the sink camera setup. They have been cropped to start at the same point in both videos.

In this test, both Yasmine and Kate collectively add 3 dirty dishes: a bowl, a fork, and a spoon. After these have been added, Yasmine washes and removes the fork and the spoon. Then, Kate washes the bowl.
